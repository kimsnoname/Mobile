## 한 줄 소개
Compose로 만들고 싶었던 가상화폐 모의투자 앱

## 기능 소개
1. 실시간 코인 호가 및 정보 확인
2. 지정가 및 시장가 매수/매도
3. 코인 가격 및 거래량 차트 확인
4. 차트 메인 지표 확인 ex) 이동평균선, 볼린져밴드, 등
5. 투자내역 포트폴리오 확인
6. 가상 KRW 입출금
7. 라이트/다크 모드 지원
